Kinectを利用したヒューマンインタフェースMIDIシーケンサ

オルゴールは，円柱上のピンに音楽情報が載せられており，それらが回ることで音楽が奏でられます．中には音楽に合わせた装飾が施されるなど，聴覚だけでなく視覚的にも楽しまれているものがあります．本作品は音楽を聴覚的・視覚的に楽しむことが出来るオルゴールの特性に着目し制作した空間呈示型オルゴールです．Kinect を用いて空間情報を音楽情報に変換し，その様子を映像として投影することで，空間の可聴化，音楽の可視化を行っています．

Flash

2018.07

Keywords / Interactive Art, Human Interface, Lighting, Projection, openCV

Tools / openFrameworks, Kinect v1

人の動きに連動した光で感情を表現したインタラクティブアート

openFrameworksとKinect v1を用いて人の動きにインタラクションする光を表現した作品．addonは，ofxBox2d，ofxFlowTools，ofxGui，ofxKinect，ofxOpenCvを引用しています．光を基調とした表現であるため明度と彩度に気を使い，カラーリングには，RGBではなくHSBを使用しています．

Face Vocoder

2018.06

Keywords / Interaction, Face Tracking, Music, openCV

Tools / openFrameworks, MIDI, Reason(DAW)

フェイストラッキングとボコーダーを組み合わせた楽器

ボコーダーという声をマイクにかけ，ストリングスやシンセなどの楽器音に置き換えて鳴らす楽器をモチーフに，openFrameworksでフェイストラッキングを行い，母音の口の形を記憶させた．「あ」という口をして，キーボードを打ち込むとその音程で「あ〜」と鳴らしてくれる楽器．声は出していないのに，ボコーダーみたいになる作品
